## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Have Nest Cli installed

```
npm i -g @nestjs/cli
```

## Database collection

```
docker-compose up -d
```

## Rebuild database

```
http://localhost:3000/api/v2/seed
```

## Stack

- MongoDB
- Nest Js
